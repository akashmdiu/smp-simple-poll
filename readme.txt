The Simple Poll is a voting poll system into your post, pages and everywhere in website by just a shortcode. Add poll system to your post by placing shortcode.

== Check Demo == 
[Demo] (https://akash.themeix.com/simple-poll/)

== Plugin URL ==
[PluginUrl] (https://wordpress.org/plugins/smp-simple-poll/)

== HERE ARE SOME FEATURES THAT COME WITH THIS SIMPLE POLL PLUGIN: ==

1. The plugin helps to place vote / poll in your website.
2. You can place user vote / poll system to your blog posts, pages and everywhere.
3. You can use shortcode to place poll / election voting system anywhere.
4. Awesome Advance UI 
5. Check each and every poll’s result.
6. Add Poll via BLOCK editor.
7. Add Poll via widget in sidebar.
8. Easy to place shortcode now having more options [SIMPLE_POLL id=”Your Poll ID”][/SIMPLE_POLL] or [SIMPLE_POLL id=”Your Poll ID”]
